# AI Voice RTC Backend

This is my backend project for AI voice calling.

Right now it can:
- do realtime voice talking
- connect android app with websocket
- use vosk speech recognition
- store calls with supabase
- basic call accept and reject

Still working on many things and code is not fully cleaned yet.

Need to add:
- better AI replies

- notifications maybe
- improve UI side

Made using:
- Python
- FastAPI
- Supabase
- Vosk
